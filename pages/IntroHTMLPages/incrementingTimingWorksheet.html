Incrementing and Timing Worksheet

Printable Version

1. Click Counter. Write an app with a single button. The button should have text of 0 to start. Each time it is clicked, increment the Button.Text property so that it displays the number of times it has been clicked.

2. Timer App. Create an app with a label. Place a 10 in the label (Label1.Text).

  1. Code the app so that it counts down from 10 to 0, changing the number in the label every second, and speaking the number. When it hits 0, the counter should stop and the app speak out the words, 'Finito!' Hint: Use a Clock component and its Clock.Timer event.
  2. Modify the app so that it counts down from 30 to 0, but only shows changes (and speaks) every five seconds (30-25-20).
    Hint: Modify the Clock.TimerInterval property and how much you decrement the value in the label each time Clock.Timer is triggered.
