Customization Exercises

1. Write a version of No Text While Driving that lets the user define custom responses for particular incoming phone numbers. For instance, if your best friend texts you, you respond with one message, but for anyone else you respond with another message. You'll need to add conditional (if) blocks that check for the particular number(s).

2. Write a version of No Text While Driving that sends custom responses based on whether the user is within a certain latitude/longitude range. So, for instance, if the app determines that you're at home, it sends back a response “At Home”.

3. Write a version of No Text While Driving that sounds an alarm when a text is received from a number in a "notify" list.

4. Write an app that texts a message to all the numbers in a list, but allow the user to enter and change the numbers for the list and modify the message sent out. Your data should be persistent.
