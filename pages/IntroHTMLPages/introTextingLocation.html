In this module, you’ll learn how to create social, location-sensitive apps that use some high-tech components including the Texting, LocationSensor, and WebView components. You’ll build a “Missing You” app that texts a list of friends with one click, a “No Texting While Driving” app that auto-responds to text messages, a “Where Am I?” app that identifies your location and shows it on a map, and you’ll be exposed to persistent (database) data and the Web Viewer component.

{% endblock %} {% block card %}

Learning Objectives

  • Learn to build apps that send texts and process incoming texts.
  • Understand how to create a list of data and process its items using a for each loop.
  • Understand how to use GPS to obtain the device’s location information.
  • Learn the basics of persistent data and the TinyDB component.
  • Learn how to show web pages, including maps, within an app.
  • Understand the basics of how URLs work and how to build them programmatically to show dynamic information (e.g., a map of your current location).
