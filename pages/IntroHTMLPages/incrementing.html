Explained

The lighter green block labeled “ScoreLabel.Text” is a “Get” block. “get” means to go look in a memory cell and see what is there. The first time the button is clicked, the “get” returns the initial value of 0, and runs the program of 0+1=1. 1 is then set into ScoreLabel.Text (and 1 appears in the user interface). The second time the button is clicked, the current value of ScoreLabel.Text is 1, so 1+1=2 is placed into ScoreLabel.Text.
If you want the number to decrement, just change + to -.
