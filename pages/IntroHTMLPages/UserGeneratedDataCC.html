Customizations

1. The NoteTaker app in the tutorial shows the items with the most recently entered at the top. Modify the code so that the notes are listed with the least recently entered at the top

 

2. Add a Textbox for the user to enter a title for each note. Then add the title as part of each note so that each note has the title for the note, the text for the note, and the timestamp. You’ll need to modify both the SubmitButton.Click event handler and the “listAsText” procedure so that the title appears with each note.
