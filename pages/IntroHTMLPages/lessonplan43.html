Creating Mobile Apps

Lesson Plan 4.3: Quiz Tutorial

Printable Version

Objectives

  • Gain experience using lists and list iteration in a more complex app
  • Learn some text processing blocks

1. Quiz Tutorial (45-60)

Introduce the quiz app by:

  • Discussing the wide variety of study guide, survey, and fun apps that can be built using what they’ll learn.
  • Discussing how it is like the Slideshow app, but now each slide is a question and the app has to evaluate the answers. But the Slideshow app was the difficult part!

If you prefer more directed tasks:

Assign the students the task of building the President’s Quiz app. The video tutorial is at http://www.appinventor.org/presidentsQuiz2-steps. A text tutorial is at: http://www.appinventor.org/PresidentsQuiz2

If you want your students to gain more experience building without explicit instructions:

Assign the students the task of creating a quiz app with questions/media of their choice, and using the Slideshow app as a starting point. You can give them the following requirements:

  • At least three questions
  • Use media (pics/video/sound) in some way
  • Provide some way for user to answer questions and see results.

2. Quiz Customizations (30-90)

  • Instead of just showing images for each question, try playing a sound clip or a short video. With sound, you can turn your quiz into a “Name That Tune” app.
  • The quiz is very rigid in terms of what it accepts as a valid answer. There are a number of ways to modify this. One way is to make the answer checking  case-insensitive--so “roosevelt” will work for the answer “Roosevelt”. Another is to use the text.contains block to see if the user's answer is contained in the actual answer. Another option is to provide multiple answers for each question, and check by iterating (foreach) through them to see if any match.
  • Difficult: Transform the quiz so that it is multiple choice. You'll need an additional list to hold the answer choices for each question. The possible answers will be a list of lists, with each sublist holding the answer choices for a particular question. Use the ListPicker component to allow the user to choose an answer.
