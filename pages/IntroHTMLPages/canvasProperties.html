Changing the Properties of the Canvas Component

When you drag a Canvas Component onto the screen, it starts out really small because its current size properties are set to automatic.


Automatic means that the size of the Canvas will change in regards to what is inside the canvas component (Similar to the horizontal arrangement).

You should start out by changing the width of the canvas to "Fill parent" (or however big the device's screen is) and change its height to around 300 or another static value that would allow the Canvas to take up most of the screen.
