In this module, you’ll learn about drawing, animation, and game-making and you’ll become familiar with the Canvas, ImageSprite, and Clock components. You’ll build a Painting app and two classic arcade games, MoleMash and Pong.

{% endblock %} {% block card%}

Learning Objectives

  • Understand how to build apps that let the user draw on a canvas.
  • Learn that a variable is used to remember information, and how to create and use variables in App Inventor.
  • Understand how to program an app to do Math, e.g., incrementing a score.
  • Learn the basics of animation and how to use the Clock component and ImageSprite properties to animate objects.
  • Learn the basics of coding randomness, e.g., an object appearing in a random place.
  • Become familiar with the Canvas, ImageSprite, and Clock components.
