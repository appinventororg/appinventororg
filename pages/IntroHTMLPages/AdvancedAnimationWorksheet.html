Shooting bullets

Printable Version

Consider the incomplete game to the right. The arrow is named ShooterImage and the ball is BulletBall. The buttons are named RotateButton and ShootButton respectively. Sketch or code blocks for the following and clearly specify the properties that should be set to start.

1.  Program the RotateButton so that it rotates the shooter 90 degrees to the right.

2.  Program the ShootButton so that it causes BulletBall to appear at the nose of the shooter, shoot out from the nose, and disappear when it hits an edge. Assume there is a way to move the shooter so it is not stationary.

3. Suppose there were three bullets and the game allowed all three to be shot and flying through the canvas at the same time. How would you program this?
