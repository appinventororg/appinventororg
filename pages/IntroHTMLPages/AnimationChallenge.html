Topics

Animation, frames-per-second, clock timers, sprite timers, speed

Coding Challenge 1: Basic Animation using a Clock Component

For these challenges, you’ll need a Canvas component that spans the width of the screen and is about 300 pixels in height. Add a button labeled “GO” below it. You’ll create two separate apps:

Move left, down, and diagonal using a Clock. Add a Clock component and three Ball components within the canvas. When the Go button is clicked, the balls should move smoothly across the canvas (you can get smooth movement by setting the Clock.TimerInterval to around 50 ms). One ball should move down, one should move left, and one should move diagonally. The balls should stop when they hit the edge of the canvas. For this challenge, use a Clock component and do not use the Ball component’s internal animation properties (Don’t modify the ImageSprite speed/heading/interval properties). Try this challenge on your own. If you need help, see: http://www.appinventor.org/TimedActivity

Coding Challenge 2: Basic Animation using Sprite Animation

Move left, down, and diagonal using Sprite animation. For this challenge, you’ll repeat the behavior of (1) but without a Clock component. Instead, use the Ball component’s internal animation properties-- Speed, Interval, and Heading.

Start by doing a SaveAs of your Coding Challenge 1 Project. Then delete the Clock component-- you won’t use it. Your goal is, once again, to have the three balls move smoothly across the canvas (one down, one left, and one diagonally). Try this challenge on your own. If you need help, see >http://www.appinventor.org/sprites

Coding Challenge 3: Back and Forth using Clock Animation

For this challenge, SaveAs your clock animation app from Challenge 1 and create a new app for the following behaviors

1. When a button is clicked, have ball 1 move back and forth across the screen, from the left edge to the right edge, and back, forever.

Try this on your own. If you need help see http://www.appinventor.org/Variables

2. When a button is clicked, have ball 2 move back and forth, from the left edge of the screen to the middle of the screen and back, forever.

Coding Challenge 4: Back and Forth using Sprite Animation

For this challenge, SaveAs your sprite app from Challenge 2 and create a new app for the following behaviors:

1. When a button is clicked, have ball 1 move back and forth across the screen, from the left edge to the right edge, and back, forever.

2. When a button is clicked, have ball 2 move back and forth, from the left edge of the screen to the middle of the screen and back, forever.
