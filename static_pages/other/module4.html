Module 4  Data

This module first introduces students to lists with concrete data, using a quiz app as an example. The module then introduces apps with user-generated data, using a notetaking app and music recorder/player as examples. The module culminates with the Broadcast Hub, an app that combines user generated data with SMS texting and complex conditionals. Note that this module is a major conceptual leap forward as the introduction of user generated data leads to much more abstract apps.

Programming concepts: lists, iteration, user generated data, more persistence

Components: Labels, Buttons, Clock, Sound, Texting

  • Quiz App: Build, Conceptualize, Customize
  • Programming Lists (ch 19, up to 273)
