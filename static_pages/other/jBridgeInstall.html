You need an interactive development environment (IDE) to build Android apps, either Android Studio or Eclipse with the Android Development Tool (ADT) plugin. For either, you'll need Java 8  so check your computer to see if the Java 8 SDK is installed (you can check on a terminal with "java -version". If you don't have Java 8 (Java 1.8.x), install it.

Install one of the following:

Android Studio

Follow the instrutions at install Android Studio 

Eclipse and ADT

1. Install Eclipse

2. Install the Android SDK

3. Within Eclipse, install the Android Development Tools (which is an Eclipse plug in).
