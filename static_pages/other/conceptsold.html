Concept-Map

Want to learn about a particular programming concept or App Inventor component? This page provides a map for getting to what you want.

Iteration

  • foreach and while: App Inventor book chapter on Iteration
  • foreach: The Notetaker app uses a foreach to display a list of notes, see steps 3-5.
  • foreach and forrange: Math Blaster App The MathBlaster app generates addition questions for students and checks their answers. It uses for range to generate a random number of items in the equation, and foreach to sum the numbers up. This tutorial is for App Inventor 2.

Persistence-- Database data

  • App Inventor book chapter on persistent data
  • No Text While Driving, step 4, provides a simple example of persistence with TinyDB.
  • Android, Where's my Car?, p. 108, provides another simple example of persistence with TinyDB.
  • Notetaker, step 5, provides a sample for saving a list to TinyDB.
