Module 2  Games

The second module introduces games and sprite animation. Students learn about timer events, incrementing a number (score), animation events like CollidedWith and conditional (if) blocks.

Programming concepts: timer events, sprite animation, incrementing variables, conditionals, complex formulas

Components: Canvas, ImageSprite, Clock, Button, OrientationSensor

  • MoleMash: Build, Conceptualize, Customize
