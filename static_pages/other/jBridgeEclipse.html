Install Eclipse and ADT on your Computer

Eclipse is an interactive development environment (IDE). It provides auto-complete, a debugger and other tools for making programming easier.

  1. Download Eclipse. You can use this to build Java programs.
  2. To build Android apps, you need an Eclipse plug-in called the Android Development tools. You can install using these instructions

You'll also need the newest version of the Android SDK library and Java 8 on your computer.
