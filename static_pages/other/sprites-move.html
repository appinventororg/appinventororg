An ImageSprite's speed, pixels/second, is actually determined by two properties. The Interval property specifies how often the sprite will be moved-- it is similar to the TimerInterval property of the Clock component. The Speed property specifies how many pixels the sprite will move each interval (the property is misnamed). In this example, the sprite moves 5 pixels per 50 ms, or 1 pixel per 10 ms, which is 100 pixels/second.

A Sprite's Heading property specifies the direction it moves. As shown below, a Heading of 270 degrees will move the sprite down.
